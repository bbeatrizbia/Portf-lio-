    • Identificador de idade

      Desenvolvi um site interativo que calcula a idade do usuário e exibe o gênero com uma imagem ilustrativa, com base nas informações inseridas de gênero e ano de nascimento. Utilizando HTML, CSS e JavaScript, criei uma aplicação simples, mas envolvente, que criar experiências personalizadas para o usuário. O site conta com um formulário onde o usuário insere seu gênero e ano de nascimento. Ao enviar essas informações, o site calcula automaticamente a idade do usuário e exibe uma imagem representativa do gênero escolhido, tornando a experiência mais visual e divertida.
